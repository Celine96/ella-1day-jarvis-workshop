# Routine: weekly-review

## Trigger

- **Cron**: `0 20 * * 0` (일요일 20:00 KST)
- 시간 변경: cron 식만 수정
- 월요일 아침으로 옮기려면: `0 7 * * 1`

## 실행 흐름

1. 페르소나 `.claude/agents/자비스.md` 로드
2. `/weekly-review` 운영 커맨드 실행 (출력 캡처)
3. Gmail 자기발송:
   - 수신자: 요한님 본인 메일 주소
   - 제목: `[자비스] 다음 주 — 아이젠하워 매트릭스 정리`
   - 본문: `/weekly-review` 출력 결과

## 안전장치

- 메일 발송 실패 시 콘솔 에러 출력 + 재시도 1회
- 다음 주 일정 0건일 경우: 일정 비어 있음 명시 + "다음 주 채울 우선순위 검토" 메시지

## 권한

- Gmail 쓰기 권한 (routine 발송용)
- Calendar 읽기 권한 (다음 주 일정 조회)

## 학습자 커스터마이징

- **시간 변경**: `0 20 * * 0` → 원하는 시간
- **월요일 아침으로**: `0 7 * * 1`
- **일시정지**: `/3-5-handoff` 참조
