`.claude/agents/{{JARVIS_NAME}}.md`를 읽고 다음을 점검해 출력합니다.

## 점검 항목

| 항목 | 체크 |
|---|---|
| §1 정체성 | 자비스 이름 박혀 있는지 |
| §2 회사 컨텍스트 | 1~2줄 명확한지 (너무 추상적이면 ❌) |
| §3 톤 | 한 줄로 명시되어 있는지 (`[학습자 작성]` 잔재 ❌) |
| §4 Q1 규칙 | 구체적 (예: "투자자 메일") 인지, 추상적 (예: "중요한 일") 아닌지 |
| §4 Q4 규칙 | 구체적인지 |
| §5 보고 형식 | 한 줄로 명시 |
| §6 시간 블록 정책 | 본인 라이프스타일과 맞는지 (질문) |
| §7 발송 채널 | "메일 자동 발송 절대 금지" 박혀 있는지 |

## 출력 형식

```
✅ 잘 채워진 섹션
- §1 ...
- §3 ...

⚠️ 보강 필요
- §2 회사 컨텍스트가 "테크 회사" 정도로 너무 추상적입니다. 어떤 단계인지 한 줄로 보강해주세요.
- §4 Q1 규칙 #2가 "중요한 메일"로 추상적. 구체적 키워드(투자자/이사회 등)로 바꿔주세요.

수정하시려면 자비스에게 대화로 말씀해주세요. 예: "Q1 규칙 #2를 '투자자 메일'로 바꿔주세요."
```

## 다음 단계 안내

리뷰·보강이 끝나면 다음 안내를 출력합니다:

> "페르소나 보강 완료. 풀 페르소나(§1~§7)가 모두 채워진 상태에서 자비스 인사를 한 번 더 들어보시죠. `/2-1-persona-greeting`을 입력해주세요."
