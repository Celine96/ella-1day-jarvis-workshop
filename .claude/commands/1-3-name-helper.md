---
description: 자비스 이름 후보 3개 제안
---

학습자의 회사 컨텍스트와 톤 선호를 고려해 자비스 이름 후보 3개를 제안합니다.

## 입력 (대화로 수집)
- 회사 한 줄 소개 (사전 체크리스트에서 가져오거나 묻기)
- 선호 톤: 정중·친근·격식 중 하나
- 한글/영문/혼합 선호

## 출력 형식

```
요한님의 자비스 이름 후보:

1. [이름] — [추천 이유 1줄]
2. [이름] — [추천 이유 1줄]
3. [이름] — [추천 이유 1줄]

마음에 드는 이름을 선택하시거나, 본인 아이디어가 있으시면 알려주세요.
```

## 작명 가이드 적용
- 짧게 (2~4자)
- 부르기 좋게
- 비서·집사·동반자 컨셉 우선
- 본인 컨텍스트 반영 OK (회사명 변형, 취향)
