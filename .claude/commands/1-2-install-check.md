description: 클로드 코드 설치·버전 검증
---

다음을 차례로 확인해 출력하세요:

1. 클로드 코드 버전 (`claude --version` 또는 환경변수 기반)
2. OS / 셸
3. `.claude/` 폴더 존재 여부
4. `.claude/agents/` `.claude/commands/` `.claude/routines/` 하위 폴더 존재 여부

✅ / ❌ 표시로 결과를 출력하고, 빠진 항목이 있으면 어떻게 보충할지 한 줄로 안내합니다.

## 다음 단계 안내

검증이 완료되면 다음 안내를 출력합니다:

> "환경 준비 완료. 1-3 챕터로 가서 자비스에게 이름을 지어줄 차례예요. `/1-3-name-helper`를 입력해주세요."
