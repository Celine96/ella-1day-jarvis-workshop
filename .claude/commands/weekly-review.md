---

호출 시점: 주말 또는 일요일 저녁. 자동 routine으로 일요일 20:00 발송 (학습자가 변경 가능).

## 1. 지난 주 회고

다음 데이터 수집:
- Calendar 지난 7일 일정 — 진행한 미팅·취소·미참석
- Gmail 발신함 — 보낸 답장·발송 메일 카운트
- 액션아이템 처리율 (가능하면)

회고 항목:
- 완료한 Q1 항목
- 미완료 Q1 항목 (다음 주 carry-over)
- 시간 사용 패턴 (회의 vs 집중작업 비율)
- 일정 충돌·과부하 발생 여부

## 2. 다음 주 미리보기 + 아이젠하워 분류

- Calendar 다음 7일 일정 전체 조회
- 페르소나 Q1·Q4 규칙 적용해 분류
- 미분류 항목은 LLM 판단

## 3. 시간 블록 추천 (주간)

- 매일 Q1·Q2 집중 시간대 1~2시간 확보
- 회의 밀집일 식별 → 다음 주 부하 분산 추천
- 사외이사·투자자 미팅이 있는 날 표기

## 4. 출력 형식

```
[자비스] 다음 주 — 아이젠하워 매트릭스 정리

자비스가 지난 주를 회고하고 다음 주 일정·할 일 (총 N건)을 정리했습니다.

## 지난 주 회고
- ✅ 완료한 Q1: N건
- ⚠️ 미완료 Q1 (다음 주 carry-over): [항목들]
- 📊 시간 사용: 회의 N시간 / 집중작업 N시간 / 비율 R%

## 다음 주 (월~일)

🔴 Q1 — Do (이번 주 안에 처리)
• [요일] [항목]

🟠 Q2 — Schedule (시간 블록 확보)
• [요일] [항목] — 추천 시간 블록: [언제]

🟡 Q3 — Delegate
• [항목] → [위임 대상]

⚪ Q4 — Delete
• [항목]

## 추천 시간 블록 (주간)
- 월 09:00~11:00 [집중 작업명]
- 화 14:00~15:30 [전략 검토]
- ...

## 다음 주 부하 진단
- 화·수 회의 밀집 (각 6건) → 목요일 집중 시간 확보 권장
- 일정 충돌: ⚠️ 수 14:00 (외부 미팅 vs 임원 콜)
```

## 원칙

- 회고는 짧게 (3~5줄), 미래가 본질
- carry-over 항목은 다음 주 Q1·Q2로 자동 격상
- 시간 블록은 페르소나 §6 정책 준수
- 자동 routine으로 일요일 20:00 발송 (학습자 지정 시간으로 변경 가능)
