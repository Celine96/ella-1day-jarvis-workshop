`.claude/commands/meeting-recap.md`를 학습자와 함께 검토하고 첫 실행으로 안내합니다.

## 흐름

1. 파일 §1~§3 짧게 설명
   - 자비스는 회의록을 만들지 않음 — 외부 도구 결과를 입력으로 받음
   - 액션아이템을 아이젠하워로 분류 (Q1·Q2 본인 / Q3 위임 담당자 추천 / Q4 폐기)
   - Q3는 담당자 추천만 — 위임 메일은 본인이 직접 작성·발송
2. 첫 실행:
   - Tiro 사용자: `/meeting-recap "[회의명]"` 으로 자동 가져오기
   - 다른 도구 사용자: `/meeting-recap` 후 회의록 텍스트 붙여넣기
3. 결과 검토 — Q1 본인 마감 · 미해결 질문 답변 일정 확인

## 자주 나오는 튜닝

- "위임 담당자 추천이 부정확" → 페르소나 §1에 팀 구성·외부 자문 명단 추가
- "결정사항과 액션아이템 구분이 모호" → 페르소나 §1에 "결정사항 = 회의에서 합의된 것 / 액션아이템 = 누군가가 그 후 실행할 것" 명시

## 다음 단계 안내

`/meeting-recap` 첫 실행 + 결과 검토가 끝나면 다음 안내를 출력합니다:

> "회의록 정리 완료. 이제 본인 옵션 채널 1개를 자비스에 추가할게요. `/3-3-connect-option-channel`을 입력해주세요."
