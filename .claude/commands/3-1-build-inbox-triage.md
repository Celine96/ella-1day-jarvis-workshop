`.claude/commands/inbox-triage.md`를 학습자와 함께 검토하고 첫 실행으로 안내합니다.

## 흐름

1. 파일 §1~§4 짧게 설명
   - §1 메일 수집 (최근 7일 미답)
   - §2 아이젠하워 분류
   - §3 Q1 답장 초안 일괄 작성 → 임시보관함 저장
   - §4 출력 형식
2. 학습자에게 첫 실행 안내:
   ```
   /inbox-triage
   ```
3. 결과 검토 + Gmail 임시보관함 확인 안내
4. Q4 항목의 의미 강조 — "이 N건은 안심하고 무시하셔도 됩니다"

## 자주 나오는 튜닝

- "Q4가 너무 많아요" → 페르소나 §4 Q4 규칙 다듬기 ("이 발신자는 항상 Q4")
- "답장 초안 톤이 어색해요" → 페르소나 §3 톤 + §5 보고 형식 다듬기
- "Q3 위임 추천이 부정확" → 페르소나 §1에 팀 구성·역할 한 줄 추가

## 결정 비용 사라지는 순간

학습자에게 "지금 미답 메일을 10분 안에 처리할 수 있는 상태가 됐어요. 이게 자비스의 진짜 가치입니다." 안내.
