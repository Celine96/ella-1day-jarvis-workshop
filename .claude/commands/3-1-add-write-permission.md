학습자에게 Gmail 쓰기 권한 추가 절차를 안내하고 검증합니다.

## 절차 안내

1. Google 계정 보안 페이지로 이동:
   `https://myaccount.google.com/permissions`
2. "Google에 액세스할 수 있는 서드파티 앱" → 클로드 코드 선택
3. **"메일 보내기"** 권한 추가 동의
4. 클로드 코드에서 Gmail MCP 재인증
5. 학습자에게 "재인증 완료되면 알려주세요" 대기

## 검증

학습자가 완료를 알리면:
```
/2-2-mcp-test
```
실행. 출력에 **"✓ Gmail 쓰기 권한 활성"** 한 줄이 추가되어야 함.

## 안전 안내

::: warning 권한 != 자동 발송
쓰기 권한이 켜졌어도 자비스는:
- 임시보관함에만 저장 (`/inbox-triage`, `/meeting-recap`)
- 본인 자기발송만 (자동 routine)
- 외부 자동 발송은 페르소나 §7에서 금지

이 경계는 워크숍 종료 후에도 유지됩니다.
:::
