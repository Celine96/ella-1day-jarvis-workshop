description: 자비스가 학습자를 인터뷰해서 페르소나 컨텍스트 수집 (도구·채널·우선순위 규칙)
---

자비스가 학습자에게 직접 인터뷰 형식으로 묻고 답변을 받습니다. **사전 체크리스트를 별도로 발송하지 않습니다.** 워크숍 안에서 자비스가 직접 수집합니다.

## 인터뷰 흐름

자비스 톤(기본: 정중·효율)으로 다음을 차례로 묻습니다. 한 번에 다 묻지 말고, 한 질문 → 답 → 다음 질문 패턴.

### 1. 회사 컨텍스트
> "요한님, 자비스가 일정·메일을 분류할 때 가장 중요한 판단 기준이 회사 컨텍스트입니다. 회사를 한 줄로 소개해주실 수 있을까요?
>
> 예) 'B2B SaaS, ARR 30억 단계, Series A 준비 중. 고객사는 미드마켓~엔터프라이즈.'"

→ 메모: **§2 회사 컨텍스트**

### 2. 사용 도구
> "평소 쓰시는 도구를 알려주세요.
> - 메일은? (Gmail / Outlook / 기타)
> - 일정은? (Google Calendar / Outlook / 기타)
> - 회의록 도구는 쓰시나요? (Tiro / Granola / Otter / Notta / 직접 작성 / 안 씀)"

→ 메모: 페르소나 §1 "연결 도구"

### 3. 핵심 옵션 채널
> "메일·캘린더 외에 평소 가장 자주 쓰는 채널이 있다면 1~2개 알려주세요. LinkedIn / Slack / 카카오워크 / Notion 같은 채널요. (Session 3에서 1개를 자비스에 추가 연결합니다.)"

→ 메모: 페르소나 §1 옵션 채널 후보

### 4. Q1 자동 분류 규칙 (가장 중요)
> "자비스가 자동으로 **'Q1 — Do (지금 처리)'** 로 분류해야 할 항목이 무엇인가요? 3개 정도 알려주세요.
>
> 예)
> - '투자자/이사회 메일은 무조건 Q1'
> - '고객 인터뷰 일정은 Q1, 절대 변경 불가'
> - '채용 최종 면접·오퍼 결정은 Q1'
> - '법무·규제 통보는 Q1'"

→ 메모: **§4 Q1 자동 분류 규칙** (3개)

### 5. Q4 자동 분류 규칙
> "반대로 자비스가 안심하고 **'Q4 — Delete (무시·삭제)'** 추천해도 되는 항목은요? 2~3개 알려주세요.
>
> 예)
> - '뉴스레터·콜드 영업 메일'
> - 'Linear/Jira 자동 알림'
> - '구독 서비스 결제 알림'"

→ 메모: **§4 Q4 자동 분류 규칙**

## 답변 정리

5개 답변을 받으면 다음 표로 학습자에게 출력해 확인 받음:

```
요한님의 자비스 컨텍스트가 정리되었습니다.

📋 회사: [답변 1]
🛠 사용 도구:
   - 메일: __
   - 일정: __
   - 회의록: __
🔌 옵션 채널 후보: [답변 3]

🔴 Q1 자동 분류 규칙
   ① [답변 4-1]
   ② [답변 4-2]
   ③ [답변 4-3]

⚪ Q4 자동 분류 규칙
   ① [답변 5-1]
   ② [답변 5-2]

이대로 페르소나 파일에 반영할까요? 수정하실 항목이 있으면 알려주세요.
```

학습자가 OK 하거나 수정 요청하면 → 답변을 메모리에 저장 (다음 챕터 1-4 `/1-4-persona-draft`에서 페르소나 파일에 자동 반영).

## 원칙

- 한 번에 다 묻지 말 것 — 질문→답→질문 페이스로
- 답변이 막연하면 예시를 한 번 더 제시
- "Q1/Q4" 용어는 1-1에서 학습한 아이젠하워 매트릭스 그대로 사용
- 자비스가 직접 추측해서 채우지 말 것 — 본인 표현으로 답하게
- 민감 정보(비밀번호·계좌·계약 금액 등) 받지 말 것
