등록된 routine을 즉시 1회 실행해 학습자 메일함으로 발송합니다. 워크숍 4시간의 결승점 — 0번 약속의 실현.

## 흐름

1. 기본 대상: `daily-morning-brief` (학습자가 다른 routine 지정 가능)
2. routine 파일을 그대로 실행:
   - 페르소나 로드
   - `/morning-brief` 실행 (출력 캡처)
   - Gmail 자기발송
3. 콘솔에 진행 상황 단계별 출력
4. "30초 안에 메일함을 확인하세요" 안내

## 콘솔 출력 형식

```
[자비스] daily-morning-brief 즉시 실행 중...
  ✓ 페르소나 로드
  ✓ Gmail 미답 메일 조회 (24h, N건)
  ✓ Calendar 오늘 일정 조회 (N건)
  ✓ 아이젠하워 매트릭스 분류 완료
    - Q1: N건 / Q2: N건 / Q3: N건 / Q4: N건
  ✓ Gmail 발송 완료 → 요한님@gmail.com

다음 30초 안에 메일함을 확인하세요.
```

## 검증 후 안내

학습자가 "메일 도착했어요"라고 알리면:

```
자비스 24/7 작동 시작.

이게 0번 프롤로그에서 미리 보신 자비스 메일과 같은 형태죠.
내일 아침 8시에 똑같은 메일이 또 도착합니다.
요한님은 아무것도 안 하셔도 돼요.

{{JARVIS_NAME}}가 24/7로 일하기 시작했습니다.
```

## 메일이 안 도착하면

스팸 함 확인 → Gmail 권한 → 페르소나 파일명 일치 → 페르소나 Q1 규칙 (분류 항목 0건이면 빈 메일) 순서로 디버깅.

## 다음 단계 안내

본인 메일함에 첫 메일 도착 확인이 끝나면 다음 안내를 출력합니다:

> "워크숍 4시간의 결승점에 도달했습니다. 마지막 챕터에서 권한 회수 가이드와 다음 단계 안내를 정리할게요. `/3-5-handoff`를 입력해주세요. (이 챕터는 인터랙티브 가이드라 본인이 원하실 때 호출하셔도 됩니다.)"
