페르소나 `.claude/agents/자비스.md`를 로드해 톤·우선순위 분류 규칙을 적용한다.

## 1. 데이터 수집 (빠짐없이)

다음 소스를 병렬 조회:

1. **Calendar (오늘 24h)** — `mcp__claude_ai_Google_Calendar__list_events`
2. **Gmail 미답 메일 (최근 24h)** — `mcp__claude_ai_Gmail__search_threads`로 `is:unread newer_than:1d`
3. (옵션) 연결된 할 일 도구 — Notion / Slack 등 페르소나에 명시된 채널

수집 결과를 단일 리스트로 통합. 누락 우려가 있으면 `⚠️ 누락 가능 항목` 섹션에 명시.

## 2. 아이젠하워 매트릭스 분류

페르소나 §4의 "Q1 자동 분류 규칙"·"Q4 자동 분류 규칙"을 우선 적용. 미분류 항목은 LLM 판단:

| 사분면 | 분류 기준 |
|---|---|
| **Q1 — Do** | 페르소나 Q1 규칙 매칭 / 마감 24h 이내 / 외부 약속·결정 |
| **Q2 — Plan** | 전략적 검토 / 요한님이 직접 해야 하는 작업 / 비긴급 |
| **Q3 — Delegate** | 위임 가능한 운영 항목 — 위임 대상 추천 동반 |
| **Q4 — Delete** | 페르소나 Q4 규칙 매칭 / 정보성 알림 / 콜드 영업 |

## 3. 시간 블록 추천

페르소나 §6 정책에 따라:
- 오전(09~12): Q1·Q2 중심
- 오후(13~17): 회의·커뮤니케이션
- 회의 사이 30분 버퍼

## 4. 충돌·과부하 검증

- 일정 시간 겹침 → ⚠️ 표시
- Q1 항목이 5건 초과 → ⚠️ "오늘 과부하" 경고
- 이동 시간 미반영 → 추천

## 5. 출력 형식

```
[자비스] 오늘 — 아이젠하워 매트릭스 정리

자비스가 오늘의 일정·할 일·메일 (총 N건)을 아이젠하워 매트릭스로 분류했습니다.

🔴 Q1 — Do (중요+긴급, 진행하기)
• [항목]
• [항목]

🟠 Q2 — Plan (중요+비긴급, 계획하기)
• [항목]

🟡 Q3 — Delegate (비중요+긴급, 위임 추천)
• [항목] → [추천 위임 대상]

⚪ Q4 — Delete (비중요+비긴급, 제거하기)
• [항목]

추천 시간 블록
• HH:MM~HH:MM [작업명]

⚠️ 충돌·경고 (있을 때만)
```

## 원칙

- TOP N으로 잘라내지 않는다. 전체를 빠짐없이.
- 모든 항목에 사분면 라벨 부착.
- 페르소나 규칙 우선, LLM 판단은 보조.
- 메일 자동 발송 금지 — 이 커맨드는 콘솔 출력만.
