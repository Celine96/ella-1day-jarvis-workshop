# 3-1. 쓰기 권한 + /inbox-triage

::: info Step 9 / 13 · Part 3 · ⏱ 25분
이 챕터를 마치면 — **미답 메일 50통이 사분면으로 분류되고, Q1 답장 초안이 임시보관함에 일괄 작성되어 있습니다. "이 5개만 답하시면 됩니다."**
:::

## 왜 쓰기 권한이 필요한가

지금까지 자비스는 읽기만 했어요. 다음 두 가지를 위해 **Gmail 쓰기 권한**이 필요합니다:

1. **`/inbox-triage`의 답장 초안** — 임시보관함에 저장하려면 쓰기 필요
2. **자동 routine의 자기발송** (3-4에서 등록) — 본인 메일함으로 발송하려면 쓰기 필요

::: warning 쓰기 권한 = 발송이 아니다
- ✅ 자비스가 임시보관함에 초안 저장
- ✅ 자비스가 요한님 자기 메일함으로 자기발송 (routine)
- ❌ 자비스가 외부 수신자에게 자동 발송 — **절대 금지**, 페르소나 §7에 박혀 있음
:::

## 1. Gmail 쓰기 권한 추가 (10분)
클로드 코드 터미널에서 다음 슬래시 커맨드를 입력하세요.

**[커맨드 입력]**

```
/3-1-add-write-permission
```

**기대 결과**: 자비스가 권한 추가 절차를 안내합니다.

::: tip 절차
1. Google 계정 보안 페이지 → 클로드 코드 권한
2. **"메일 보내기"** 권한을 추가로 동의 (이전 "메일 읽기"는 그대로 유지)
3. 클로드 코드에서 Gmail MCP 재인증
:::

검증:
클로드 코드 터미널에서 다음 슬래시 커맨드를 입력하세요.

**[커맨드 입력]**

```
/2-2-mcp-test
```

이전과 같은 출력에 더해서 **"✓ Gmail 쓰기 권한 활성"** 한 줄이 추가되어야 합니다.

## 2. /inbox-triage 빌드 (5분)
클로드 코드 터미널에서 다음 슬래시 커맨드를 입력하세요.

**[커맨드 입력]**

```
/3-1-build-inbox-triage
```

자비스가 `.claude/commands/inbox-triage.md`를 함께 살펴봅니다. 핵심 동작:

1. 미답 메일 50통 수집 (최근 7일)
2. 페르소나 Q1·Q4 규칙 우선 적용해 4사분면 분류
3. **Q1 항목에 답장 초안 자동 작성** → Gmail 임시보관함 저장

## 3. 첫 실행 (10분)

::: tip /inbox-triage 실행
요한님의 받은편지함을 자비스가 정리할 차례입니다.
클로드 코드 터미널에서 다음 슬래시 커맨드를 입력하세요.

**[커맨드 입력]**

```
/inbox-triage
```
:::

**기대 결과**:

```
[자비스] 받은편지함 정리 (총 47건)

🔴 Q1 — Do (답장 필수, 답장 초안 작성됨)
• [발신자] [제목 일부]
  → 임시보관함 저장 ✓
• ... (예: 5건)

🟠 Q2 — Plan (시간 확보 후 답장)
• [발신자] [제목] — 추천 처리 시간: 내일 오전
• ... (예: 8건)

🟡 Q3 — Delegate (위임 추천)
• [발신자] [제목] → 담당 추천: [팀]
• ... (예: 6건)

⚪ Q4 — Delete (제거하기 가능)
• [발신자] [제목] (사유: 뉴스레터)
• ... (예: 28건)

## 요약
- Q1 5건 답장 초안 → 임시보관함 (검토 후 발송)
- Q3 6건 위임 추천
- Q4 28건 무시 가능 — 결정 비용 사라짐
```

## 4. 임시보관함 확인

Gmail에서 **"임시보관함(Drafts)"**을 열어보세요. 자비스가 작성한 답장 초안 5건이 들어 있어야 합니다.

각 초안:
- 페르소나 톤이 적용되어 있고
- 원본 메일을 인용하며
- 요한님의 보고 형식 선호 따라 작성

::: tip 검토 → 발송
초안을 열어서 검토하시고, 마음에 들면 **"보내기"** 버튼만 누르시면 됩니다. 마음에 안 들면 자비스에게 말씀해서 다시 작성하게 하실 수 있어요.
:::

## 5. Q4 무시의 자유

`/inbox-triage`의 진짜 가치는 Q4입니다.

> **"이 28건은 안심하고 무시하셔도 됩니다."**

매일 메일함에서 30분씩 사라지던 결정 비용이 한 번에 사라져요. 요한님은 Q1·Q3만 보면 됩니다.
