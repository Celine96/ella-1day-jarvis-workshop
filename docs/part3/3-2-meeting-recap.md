# 3-2. 회의록 + /meeting-recap

::: info Step 10 / 13 · Part 3 · ⏱ 20분
이 챕터를 마치면 — **회의록을 입력하면 액션아이템이 사분면으로 정리되고, Q3 위임 항목은 후속 메일 초안이 임시보관함에 작성됩니다.**
:::

## 자비스는 회의록을 만들지 않습니다

::: tip 분업 구조
- 회의록 작성 = 요한님이 이미 쓰는 도구 (Tiro / Granola / Otter / Notta / 직접 작성 등)
- 회의록 → 액션아이템 정리 + 후속 메일 = **자비스의 일**
:::

자비스의 가치는 회의록을 만드는 게 아니라 **그 다음 단계**예요. 액션아이템을 빠짐없이 사분면으로 분류하고, 위임 메일 초안까지 작성합니다.

## 1. 본인 회의록 도구 확인 (3분)
클로드 코드 터미널에서 다음 슬래시 커맨드를 입력하세요.

```
/3-2-connect-meeting-tool
```

자비스가 사전 체크리스트 답변을 보고 입력 방식을 추천합니다.

| 본인 도구 | 입력 방식 |
|---|---|
| **Tiro** | A. MCP 직결 (자동) |
| Granola / Otter / Notta / 기타 | B. 붙여넣기 |
| 직접 작성 | B. 붙여넣기 |
| 안 씀 | 이 챕터 스킵 가능 — 회의록 도구 도입 후 다시 |

## 2-A. Tiro 사용자 — MCP 직결
클로드 코드 터미널에서 다음 슬래시 커맨드를 입력하세요.

```
/meeting-recap "어제 투자자 미팅"
```

자비스가 Tiro에서 해당 회의록을 자동으로 가져와 처리합니다.

## 2-B. 다른 도구 사용자 — 붙여넣기

회의록 도구에서 transcript를 복사한 뒤:
클로드 코드 터미널에서 다음 슬래시 커맨드를 입력하세요.

```
/meeting-recap
```

자비스가 회의록 텍스트를 받을 준비를 합니다. 그대로 붙여넣으시면 됩니다.

## 3. /meeting-recap 빌드 + 실행 (15분)
클로드 코드 터미널에서 다음 슬래시 커맨드를 입력하세요.

```
/3-2-build-meeting-recap
```

자비스가 `.claude/commands/meeting-recap.md`를 함께 살펴봅니다. 핵심 동작:

1. 회의록 파싱 (제목·일시·참석자·결정사항·액션아이템·미해결 질문)
2. **액션아이템 아이젠하워 분류**
   - Q1: 요한님 담당 + 7일 이내
   - Q2: 요한님 담당 + 8일 이상 또는 미정
   - Q3: 타인 담당 → **후속 메일 초안 작성**
   - Q4: 폐기 가능
3. Q3 항목은 **Gmail 임시보관함에 후속 메일 초안 저장**

**기대 결과**:

```
[자비스] 어제 투자자 미팅 회의록 정리

📅 일시: 2026-05-06 09:00~10:00
👥 참석자: 요한님, A 파트너, B 어소시에이트

## 결정사항
• Series A 라운드 사이즈 50억 → 70억으로 상향 검토
• 다음 마일스톤: ARR 50억 달성 (Q3 말)

## 액션아이템 (아이젠하워 분류)

🔴 Q1 — 요한님 본인이 7일 이내
• 업데이트된 IR 자료 작성 — 마감 5/12
• 핵심 고객 3명에게 레퍼런스 콜 부탁 — 마감 5/10

🟠 Q2 — 본인이 시간 확보 필요
• 70억 라운드 사이즈에 맞는 valuation 모델 재산정 — 마감 5/20

🟡 Q3 — 위임 (후속 메일 초안 작성됨)
• 데이터룸 업데이트 → 담당: 재무팀장, 임시보관함 ✓
• 법무 문서 1차 검토 → 담당: 외부 자문, 임시보관함 ✓

⚪ Q4 — 스킵
• A 파트너 명함 정리 (자동)

## 미해결 질문
• B 어소시에이트가 보낸 보안 실사 양식 — 답변 마감 5/15

후속 메일 초안 2건이 Gmail 임시보관함에 저장되었습니다. 발송 전 검토해주세요.
```

## 4. 후속 메일 검토

Gmail 임시보관함을 열어서 자비스가 작성한 후속 메일 초안을 확인하세요.

::: tip 후속 메일 패턴
- **수신자**: 위임받을 사람
- **제목**: `[{{회의명}} 후속] {액션아이템 요약}`
- **본문**: 회의 결정사항 인용 + 액션아이템 + 마감 명시 + 페르소나 톤
:::

검토하시고 마음에 드시면 **"보내기"** 누르시면 됩니다. 자비스가 자동 발송하지 않습니다.
