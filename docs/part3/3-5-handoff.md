# 3-5. 권한 회수 + 다음 단계

::: info Step 13 / 13 · Part 3 · ⏱ 10분
이 챕터를 마치면 — **워크숍 종료 후 자비스를 안전하게 운영하는 가이드와 향후 확장 방향이 손에 들어옵니다.**
:::

## 1. 권한 관리 (3분)

::: warning 자비스를 그만 쓰고 싶을 때
자비스가 본인 데이터에 접근하는 모든 권한은 **언제든지 회수**할 수 있습니다.

```
/3-5-handoff
```

자비스가 단계별 가이드를 안내합니다.
:::

회수 절차 요약:

| 권한 | 회수 위치 |
|---|---|
| Gmail / Calendar | Google 계정 → 보안 → 클로드 코드 액세스 → 제거 |
| Slack | Slack 워크스페이스 설정 → 앱 관리 → 클로드 코드 제거 |
| Notion | Notion 설정 → 연동 → 클로드 코드 제거 |

회수하시면 즉시 자비스는 본인 데이터에 더 이상 접근할 수 없게 됩니다. routine도 자동으로 데이터 부족 에러가 나면서 멈춥니다.

## 2. routine 일시정지 / 휴가 모드 (3분)

권한 회수 없이 잠깐 자비스를 쉬게 하려면:

| 상황 | 방법 |
|---|---|
| **이번 주 휴가** | 자비스에게 "이번 주 routine 정지해주세요" → routine 등록 해제, 다음 주 다시 등록 |
| **시간 조정** | 자비스에게 "평일 9시로 바꿔주세요" → cron 자동 업데이트 |
| **routine 영구 해제** | 자비스에게 "daily routine 삭제해주세요" |

## 5. 페르소나 진화 (선택)

자비스를 쓰다 보면 분류가 마음에 안 드는 경우가 종종 있어요. 그때마다 자비스에게 말씀하시면 페르소나가 다듬어집니다.

```
"이 메일은 항상 Q3로 분류해주세요. 우리 회사 운영팀 관할이에요."
"이 일정은 절대 옮기면 안 됩니다. Q1 고정으로요."
"Q4 항목은 발신자만 보여주고 제목은 생략해주세요."
```

페르소나는 살아 있는 문서예요. 한 달 쓰시면 본인에게 딱 맞는 비서가 됩니다.
