## 워크북 사용법

이 워크북은 **직접 따라하며 배우는 실습 중심** 자료입니다.

### 구조
- **[개념]** — 읽고 이해하는 파트
- **[실습]** — 직접 수행하는 파트 (슬래시 커맨드 호출)
- **[체크포인트]** — 진행 상황 확인 + 주요 마일스톤

### 소요시간
약 **4시간** (Prologue 5분 + Part 1 55분 + Part 2 90분 + Part 3 110분)

### 사전 준비물
- 노트북, 인터넷 연결
- 클로드 코드 설치 (Pro/Max 구독)
- **본인 Gmail · Google Calendar 계정** — 워크숍 안에서 자비스에 직접 연결

### 학습 모드
- 프롬프트가 완벽하지 않아도 괜찮습니다. 자비스는 **클로드 코드와의 대화로 언제든 수정**할 수 있어요.
- 모르는 부분 있으면 멈추고 물어보세요.
- 민감 정보는 페르소나에 적지 마세요. 자비스 보안 원칙 7가지는 [1-2 §5](/part1/1-2-install#5-자비스의-보안-원칙-7가지)에서 자세히.
