## 자비스 기능 미리보기

오늘 만드는 자비스는 모든 일정·할 일·메일을 **아이젠하워 매트릭스** 4사분면으로 자동 분류합니다.

**아이젠하워 매트릭스(The Eisenhower Matrix)** 는 미국 34대 대통령 드와이트 아이젠하워(Dwight Eisenhower)의 이름을 따서 스티븐 코비(Stephen Covey)가 고안한 도구로, 개인 생활과 직장 생활을 아우르는 모든 활동을 **긴급성과 중요성**에 따라 4사분면으로 분류해 우선순위를 정합니다.

![아이젠하워 매트릭스](/eisenhower-matrix.png)

🔴 **Do** 진행 · 🟠 **Plan** 계획 · 🟡 **Delegate** 위임 · ⚪ **Delete** 제거

워크숍이 끝나는 다음 날부터, **매일 아침 8시에 자비스가 본인 메일함으로 이런 메일을 보내줍니다.** 노트북을 켜지 않아도 자동으로요.

::: details 메일 본문 펼쳐 보기
```

아이젠하워 매트릭스로 분류했습니다.

🔴 Q1 — Do (진행하기)
• 09:00 투자자 미팅 — IR 자료 후속 답변 (8:30 안에 회신)
• 채용 후보자 A 최종 결정 — 이번 주 데드라인

🟠 Q2 — Plan (계획하기)
• 14:00 팀 위클리 — 결정사항 3건 공유 준비
• Q2 OKR 검토 — 금요일 전 (수요일 오후 추천)

🟡 Q3 — Delegate (위임하기)
• CFO 1Q 비용 승인 회신 → 재무팀장
• 슬랙 #product DM 3건 → 담당 PM

⚪ Q4 — Delete (제거하기)
• 뉴스레터 3건, Linear 알림 12건 (휴지통)

추천 시간 블록
• 08:30~09:00 IR 후속 메일 답장
• 13:30~14:00 위클리 자료 정리

⚠️ 일정 충돌: 11시 임원 콜과 12시 점심 30분 겹침
```
:::

## 워크숍 구성

오늘 4시간은 **3개 Part**로 진행됩니다. 자비스를 정의하고(Part 1) → 본인 데이터에 연결해 깨우고(Part 2) → 자동화로 일상화하는(Part 3) 흐름이에요.

| 단계 | 무엇을 하나 | 산출물 |
|---|---|---|
| **Part 1 — 자비스 정체성** | 자비스 작명 + 회사·우선순위 인터뷰로 페르소나 초안 자동 작성 | 자비스 페르소나<br>`.claude/agents/{이름}.md` |
| **Part 2 — 자비스 깨우기** ⭐ | Gmail·Calendar 본인 계정 연결<br>**매일 브리핑·다음 회의 준비·주간 회고**를 본인 데이터로 첫 실행 | 운영 커맨드 3종<br>`/morning-brief`<br>`/meeting-prep`<br>`/weekly-review` |
| **Part 3 — 자비스 일상화** 🎯 | **받은편지함 정리·회의록 처리** 추가<br>옵션 채널 1개 연결<br>자동 routine 등록 → 24/7 작동 시작 | 운영 커맨드 2종 (`/inbox-triage` · `/meeting-recap`)<br>+ 자동 routine 2종<br>+ 옵션 채널 1개 |

⭐ 본인 데이터로 첫 결과를 받는 순간 / 🎯 자비스가 24/7로 작동 시작

각 챕터는 **[개념] → [실습] → [체크포인트]** 패턴으로 진행됩니다. 학습용 더미 데이터(`samples/`)는 이미 준비돼 있어요 — Gmail 연결 전 워밍업용.
