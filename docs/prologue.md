# 0. 프롤로그 : 자비스, 들어보셨죠?

::: tip 5분 · 워크숍 시작 직전
오늘 4시간 동안 **무엇을 만들고, 어떤 파일들이 손에 남는지** 먼저 보고 시작합니다.
:::

요한님의 하루를 떠올려 봅시다. 메일 50통, 일정 8건, 슬랙 알림 30개, 회의록 4건이 매일 쌓입니다. **무엇에 먼저 시간을 쓸지 결정하는 비용만 매일 30분이 사라지고 있어요.**

이런 결정을 대신 정리해주는 비서가 있다면 어떨까요? 영화에는 이미 있죠. **〈아이언맨〉**의 토니 스타크에게는 비서 AI — **자비스(J.A.R.V.I.S.)** 가 있습니다.

![J.A.R.V.I.S](/jarvis.png)

토니가 슈트를 입으면 일정을 알려주고, 작전 중에는 데이터를 정리하고, 평소에는 회사 일을 거듭니다. 한마디로 **"사람처럼 대화하면서 모든 디지털 시스템을 연결해주는 초개인화 AI 비서"** 인거죠.

::: details 자비스 HUD 더 보기
![자비스 HUD 디스플레이](/jarvis-hud.png)

작전 중 토니의 시야에 모든 정보를 한 화면에 정리해 보여주는 자비스의 HUD.
:::

## 워크숍 구성

| 단계 | 무엇을 하나 | 산출물 |
|---|---|---|
| **Part 1 — 자비스 정체성** | 자비스 작명 + 회사·우선순위 인터뷰로 페르소나 자동 작성 | 자비스 페르소나<br>`.claude/agents/{이름}.md` |
| **Part 2 — 자비스 깨우기** ⭐ | Gmail·Calendar 본인 계정 연결<br>**매일 브리핑·다음 회의 준비·주간 회고**를 본인 데이터로 첫 실행 | 운영 커맨드 3종<br>`/morning-brief`<br>`/meeting-prep`<br>`/weekly-review` |
| **Part 3 — 자비스 일상화** 🎯 | **받은편지함 정리·회의록 처리** 추가<br>옵션 채널 1개 연결<br>자동 routine 등록 → 24/7 작동 시작 | 운영 커맨드 2종 (`/inbox-triage` · `/meeting-recap`)<br>+ 자동 routine 2종<br>+ 옵션 채널 1개 |

⭐ 본인 데이터로 첫 결과를 받는 순간 / 🎯 자비스가 24/7로 작동 시작

각 챕터는 **[개념] → [실습] → [체크포인트]** 패턴으로 진행됩니다. 학습용 더미 데이터(`samples/`)는 이미 준비돼 있어요 — Gmail 연결 전 워밍업용.

## 우리, 토니 스타크가 되어볼까요?

![토니 스타크와 자비스](/tony-stark.png)

토니가 자비스에게 말로 묻고, 자비스가 즉시 정리해 답하는 모습. 오늘 만드는 자비스도 그런 비서예요. 슈트나 우주는 못 만들지만, **요한님의 일정·메일·할 일을 매일 알아서 정리해주는 시간관리 비서**는 4시간 안에 만들 수 있습니다.

워크숍이 끝나는 다음 날부터, 자비스는 **매일 아침 8시에 본인 메일함으로 정리된 하루를 보내줍니다.** 노트북을 켜지 않아도요.

> 💡 자비스가 어떻게 정리하는지, 무엇을 하고 무엇을 안 하는지는 [1-1 자비스란 무엇인가](/part1/1-1-intro)에서 자세히 다룹니다.

준비되셨으면 [Part 1 — 자비스 정체성](/part1/1-1-intro)으로 가시죠.
