# 0. 오늘 만드는 자비스

::: tip 5분 · 워크숍 시작 직전
오늘 4시간 동안 **무엇을 만들고, 어떤 파일들이 손에 남는지** 먼저 보고 시작합니다.
:::

## 완성본 미리보기

오늘 만드는 자비스는 모든 일정·할 일·메일을 **아이젠하워 매트릭스** 4사분면으로 자동 분류합니다.

::: tip 아이젠하워 매트릭스란?
**아이젠하워 매트릭스(The Eisenhower Matrix)** 는 미국 34대 대통령 드와이트 아이젠하워(Dwight Eisenhower)의 이름을 따서 스티븐 코비(Stephen Covey)가 고안한 도구로, 개인 생활과 직장 생활을 아우르는 중요한 활동을 **긴급성과 중요성**에 따라 분류해 우선순위를 정하는 2x2 매트릭스입니다.

![아이젠하워 매트릭스](/eisenhower-matrix.png)

🔴 **Do** 진행 · 🟠 **Plan** 계획 · 🟡 **Delegate** 위임 · ⚪ **Delete** 제거

자비스는 회사 컨텍스트·우선순위 규칙을 받아 모든 항목을 위 4사분면에 자동 배치합니다. 1-1 챕터에서 자세히.
:::

워크숍이 끝나는 다음 날부터, **매일 아침 8시에 자비스가 본인 메일함으로 이런 메일을 보내줍니다.** 노트북을 켜지 않아도 자동으로요.

```
[자비스] 오늘 — 아이젠하워 매트릭스 정리

자비스가 오늘의 일정·할 일·메일 (총 14건)을
아이젠하워 매트릭스로 분류했습니다.

🔴 Q1 — Do (중요+긴급, 진행하기)
• 09:00 투자자 미팅 — 어제 보낸 IR 자료 후속 답변 미수신 (8:30 안에 회신)
• 채용 후보자 A 최종 결정 — 이번 주 데드라인

🟠 Q2 — Plan (중요+비긴급, 계획하기)
• 14:00 팀 위클리 — 어제 결정사항 3건 공유 준비 (13:30 30분)
• Q2 OKR 검토 — 금요일 전 (수요일 오후 추천)

🟡 Q3 — Delegate (비중요+긴급, 위임하기)
• CFO 1Q 비용 승인 회신 → 재무팀장 1차 검토
• 슬랙 #product DM 3건 → 담당 PM 패스

⚪ Q4 — Delete (비중요+비긴급, 제거하기)
• 뉴스레터 3건 (읽기만)
• Linear 알림 12건 (휴지통)

추천 시간 블록
• 08:30~09:00 IR 후속 메일 답장 (10분)
• 13:30~14:00 위클리 자료 정리

⚠️ 일정 충돌: 11시 임원 콜과 12시 점심 30분 겹침
```

이 메일 한 통으로 **매일 아침 30분의 결정 비용**이 사라집니다.

## 자비스가 하는 일

오늘 만드는 자비스의 **5가지 핵심 기능**입니다.

1. **매일 아침 자동 브리핑**
   - 일정·메일·할 일 종합 → 아이젠하워 분류
   - 시간 블록 추천 + 충돌·과부하 경고
2. **받은편지함 자동 정리**
   - 미답 메일 사분면 분류 (Q1·Q3·Q4)
   - Q1 답장 초안을 Gmail 임시보관함에 일괄 작성
3. **다음 회의 자동 준비**
   - 참석자·과거 미팅·관련 메일 분석
   - 안건 사분면 분류 + 미리 준비할 것 추천
4. **회의록 → 액션아이템**
   - Tiro·Granola·Otter 회의록 입력
   - 액션 사분면 분류 + Q3 후속 메일 초안
5. **주간 회고·다음 주 우선순위**
   - 지난 주 회고 (완료·carry-over·시간 사용)
   - 다음 주 시간 블록 추천 + 부하 분산

## 4시간 워크숍 구성

| 시점 | 단계 | 무엇을 하나 (만드는 파일) |
|---|---|---|
| 0:00 | **Prologue (5분)** | 지금 이 화면 |
| 0:05~1:00 | **Part 1 — 자비스 정체성 (55분)** | 작명 · 도구·우선순위 인터뷰 → **자비스 페르소나** (`.claude/agents/{이름}.md` — 이름·톤·우선순위 규칙·시간 블록 정책) 자동 작성 |
| 1:00~2:30 | **Part 2 — 자비스 깨우기 (90분)** | 본인 Gmail/Calendar 연결 → **운영 슬래시 커맨드 3종** (`/morning-brief` ⭐ · `/meeting-prep` · `/weekly-review`) |
| 2:30~4:00 | **Part 3 — 자비스 일상화 (110분)** | **운영 커맨드 2종** (`/inbox-triage` ⭐ · `/meeting-recap`) · 옵션 채널 1개 · **자동 routine 2종** (`.claude/routines/` — 매일 아침·주간 자기발송) 등록 🎯 |

⭐ = 본인 데이터로 첫 결과를 받는 순간 / 🎯 = 자비스가 24/7로 작동 시작

> 학습용 더미 데이터 (`samples/inbox_sample.json` 외 1)는 이미 준비되어 있어요 — Gmail 연결 전 워밍업용.

워크숍이 끝나면 위 파일들이 모두 손에 남고, 그대로 본인 일상에서 매일 작동합니다.

## 이렇게 진행돼요

각 챕터는 같은 패턴으로 구성됩니다.

- **[개념]** — 읽고 이해 (1~2분)
- **[실습]** — 슬래시 커맨드로 직접 만들기 + 본인 데이터 연결
- **[체크포인트]** — 진행 상황 확인 + 우와 모먼트
